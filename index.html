Backup e Restauração de Dados

📤 Exportar Backup

Exporte todos os dados do sistema (pessoas cadastradas e notas) para um arquivo de backup. Este arquivo pode ser usado para:

  • Fazer backup dos seus dados
  • Compartilhar dados entre dispositivos
  • Transferir dados para outro computador

📥 Importar Backup

Importe um arquivo de backup para restaurar os dados no sistema. Atenção: Esta ação substituirá todos os dados atuais.

🗑️ Limpar Todos os Dados

Remova todas as pessoas e notas do sistema. Esta ação não pode ser desfeita! Use com cuidado.
